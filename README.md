# Email ML - 垃圾邮件分类算法

基于机器学习的垃圾邮件分类系统实现，包含完整报告。

## 项目简介

本项目实现了多种机器学习算法用于垃圾邮件分类，包括：
- 朴素贝叶斯分类器
- 支持向量机 (SVM)
- 决策树
- 随机森林

## 技术栈

- **语言**: Scala
- **框架**: Spark MLlib
- **数据集**: 公开邮件数据集

## 项目结构

```
email-ml/
├── src/              # 源代码
├── data/             # 数据集
├── report/           # 完整报告
└── README.md
```

## 功能特性

- ✅ 多种 ML 算法实现
- ✅ 特征提取与处理
- ✅ 模型训练与评估
- ✅ 完整实验报告

## 快速开始

### 环境要求

- Scala 2.11+
- Spark 2.4+
- SBT

### 运行

```bash
sbt run
```

## 结果评估

| 算法 | 准确率 | F1 Score |
|------|--------|----------|
| 朴素贝叶斯 | 92.5% | 0.89 |
| SVM | 94.2% | 0.91 |
| 随机森林 | 95.1% | 0.93 |
